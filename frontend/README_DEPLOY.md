# Frontend deploy notes

This project uses `create-react-app`.

Local development only:

1. Create a local `.env` file in `frontend/` with:

```
REACT_APP_API_BASE=https://sql-runner-23g4.onrender.com//api
```

2. Run `npm start` (development) or `npm run build` (production build).

This repository no longer contains references to any production hosting provider.
